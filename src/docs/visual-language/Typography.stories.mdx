# Typography

The main typeface used within the BFI.org website is **Open Sans**. This is used for all body text and some headings.

## Headings

The only time that the typeface used is not Open Sans is on main headings (H1), when these are used the typface is **Archivo Black**, written in all caps. Writing in all caps is known to be less legible when reading at long lengths, therefore titles using all caps should be kept to a short length which will also increase SEO due to Google displaying only the first 50-60 characters of a title. **Never apply all caps to a long piece of text.**
