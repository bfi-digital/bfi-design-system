# Typography

The main typeface used within the BFI.org website is **Open Sans**. This is used for all body text and some headings.

The typeface used on main headings is **Archivo Black**, in all caps. Writing in all caps is known to be less legible, therefore titles using all caps should be kept to a short length which will also increase SEO due to Google displaying only the first 50-60 characters of a title.
