# QuickFacts

A datalist with terms and values, which can be displayed in a two-column or one-column layout.

The `facts` prop is an array of objects with `label` and `value` properties.
If the label property of any fact is falsy, the value of that fact will
span all columns and be styled with a lower font weight.

The grid uses one column by default, but can be changed to two columns by setting
the `columns` attribute to `2`. Alternatively this can be set though CSS, by setting
the custom property `--quick-facts-columns` to `2`.
