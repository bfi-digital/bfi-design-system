# HeroBasic

This is a much simpler version of the Hero component. It accepts an optional image URL as the background and allows
any content to be embeded as children. It comes with a HeroHeadline component for adding a large title, and can be
subclassed to allow any customisation needed for a specific page, and a Standfirst wrapper component.

The image URL can be empty, in which case the component will just revert to a standard sized container.
