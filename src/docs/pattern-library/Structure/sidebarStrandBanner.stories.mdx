# Sidebar Strand Banner
 
This component is automatically added to a Show page teplate if this Show page has been added to a Strand taxonomy within the BFI CMS. 

This has been built specifically for London Film Festival, however it could be expanded to be utlised for other categorisations or collections of showings. 

The Strandbanner includes a colour scheme, image, text and link. It can also have an optional sponsor.
