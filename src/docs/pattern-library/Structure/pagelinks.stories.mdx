# PageLinks
 
A full width block used to provide a list of links to other pages.

This can be used on Service list pages which provide navigation to other pages, or to provide a list of ‘related pages’ at the bottom of a page.

If linking to 3 or less pages the design of this list will set the cards to be thirds, if the list is 4 or more, the cards will be set to quarters.

Depending on the pillar that this is set to, it will have a different colour scheme.
