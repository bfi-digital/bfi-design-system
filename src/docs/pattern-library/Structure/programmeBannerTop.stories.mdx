# Programme Banner

The programme banner was created to display at the top of a feature set of pages as a way of linking the user back to a central hub.

For example within the London Film Festival, all pages within this can be added to the taxonomy for LFF. Each page will then have this banner included underneath the header - it provides context for the user to which section they are currently in, and an easy link back to the homepage of this section.
