# Headline
 
**Used for all titles and sub titles of all levels**

Use headings consistently and in a semantic hierarchy to create a clear segmentation of content throughout articles and service pages.

There should only ever be one level 1 heading on each page.

## Sizes

4 sizes are supported for headings, the hierarchy of titles should be taken into consideration and used in a semantic order:
- H1 is used only as the primary heading of the page, and only used once.
- H2 used as a main subheading which divides a section of content.
- H3 used as a sub sub heading which should be used to sub section content underneath a corresponding H2 section.
- H4 is used only as a visual flourish on cards of content, these cards will almost always be sitting under at least a section with a H2 sub heading.
