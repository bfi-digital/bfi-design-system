# Headline
 
**Used for all titles and sub titles of all levels**

Use headings consistently and in a semantic hierarchy to create a clear segmentation of content throughout articles and service pages.

There should only ever be one level 1 heading on each page.

## Sizes

4 sizes are supported for headings
