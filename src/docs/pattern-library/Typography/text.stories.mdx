# Text

This is component used to display body content text. 

On pages they have a minimal WYSIWYG editor which connects to this Text component, it allows for some basic text styling such as Bold/Italic/Underline/etc and to create inline links to internal and external urls. It also has the option for creating unordered and ordered lists.

On articles there is a more complicated WYSIWYG which includes all article slices, each block of text that within an article is seperated and displayed within a Text component like this.

This component can also be used to style tables which you can see in various states within the examples.
